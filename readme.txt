== Description ==
The Kalooga wordpress plugin is currently NOT supported.

== Installation ==
Use the Plugin Installer or:

   1. Download the plugin and unzip it
   2. Put the 'kalooga' directory into your wp-content/plugins/ directory.
   3. Go to the Plugins page in your WordPress Administration area and click Activate next to Kalooga.
   4. Have fun!

This plugin has been tested in Wordpress 2.5.1, and 2.8.4.
